# 毛府酒庄订单与CRM管理后台

支持销售、职能、管理员三角色的 B 端订单与 CRM 管理系统。

## 技术栈

- Next.js 16 + TypeScript + Tailwind CSS
- Prisma 7 + SQLite
- JWT 会话认证 + 角色权限控制

## 快速开始

```bash
cd ~/Projects/maofu-crm
npm install
cp .env.example .env
npx prisma db push
npm run db:seed
npm run dev
```

访问 http://localhost:3000

## 主要功能

- **渠道管理**：管理员配置客户渠道类型
- **客户管理**：软删除/恢复、转移销售、手机号脱敏、分页与检索
- **订单管理**：自动/手动总金额、毛利毛利率、修改审计、多维筛选
- **产品规格**：瓶/箱/件/套单位，下单时自动适配
- **数据概览**：业绩/新增/流失客户同比曲线（今年 vs 去年）
- **水墨中国风 UI**：宣纸底色、墨色文字、朱砂点缀
